<h1>Programa de Porcentagem de Pares e Ímpares</h1>
<hr> <h2>Regras do programa</h2>

Inputs inválidos (texto, símbolos, qualquer bagunça) → ignorados.

Número negativo → encerra o programa imediatamente.

Apenas números inteiros são aceitos.

<hr> <h2>Como funciona</h2>

O programa roda em um loop pedindo um valor.
Se for inteiro válido, entra nas estatísticas.
Se for negativo, o programa encerra e mostra:

Total de números válidos

Quantidade de pares

Quantidade de ímpares

Porcentagem de cada categoria

<hr> <h2>Estrutura do projeto</h2>

Main.java → lógica principal do programa

MyUtil.java → validação de inteiro usando regex

<hr> <h2>Objetivo do projeto</h2>

Esse exercício serve para treinar:

Leitura e validação de entrada

Loop com sentinela

Uso básico de regex

Contadores e porcentagens

Organização simples entre classes

<hr> <h2>Status atual</h2>

Projeto simples, funcional e perfeito pra prática de iniciante.
Sem gambiarra que compromete lógica, só o essencial.
